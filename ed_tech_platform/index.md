# 学校のICT化をサポートするSaas型プラットフォーム

## サービスについて
* 2019年5月時点で2500校の学校(全国の高校の2校に1校)が導入
* ユーザー数116万人

## 技術スタック
アプリケーションレイヤーは機能毎に分かれ、データベースは共通のものを参照している、"分断されたモノリス"構成で1つのサービスを構築しています。
その為各ソフトウェアのバージョンは使用していた中での最新バージョンのみ記述します。
